# Cables

## Straight

Es para conectar:

|Inicio|dest|
|:-:|:-:|
|Router| Switch|
|Router| Hub|
|Switch| Computadora|
|Computadora| Hub|

## Crossover

Es para conectar:

|Inicio|dest|
|:-:|:-:|
|Computadora| Computadora|
|Computadora| Router|
|Router| Router|
|Switch| Switch|
|Hub| Hub|

## Grupos

|1|2|
|:-:|:-:|
|Computadora|Switch|
|Router|Hub|

Para conectar un dispositivo de diferente grupo, usas **straight**

Si es del mismo grupo, usas **crossover**
